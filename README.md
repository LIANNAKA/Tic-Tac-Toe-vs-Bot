🎮 Tic Tac Toe vs AI Bot

🔗 Live Demo: https://tic-tac-toe-vs-bot.onrender.com

An interactive web-based Tic Tac Toe game powered by an intelligent AI opponent built using the Minimax Algorithm. This project demonstrates core concepts of game theory, decision-making, and backend integration using Python.

🚀 Project Overview

This application recreates the classic Tic Tac Toe game with an AI-driven opponent that plays optimally. The project was developed to gain hands-on experience with:

Algorithmic problem-solving
AI decision-making techniques
Full-stack development and deployment

The AI ensures a challenging experience by evaluating all possible moves and selecting the most optimal one in real-time.

🧠 Key Highlights
♟️ Unbeatable AI Logic using Minimax Algorithm
🔄 Real-time Gameplay Interaction between frontend and backend
🌐 Deployed Web Application accessible globally
🧩 Clean separation of UI and logic (Frontend + Flask API)
⚡ Efficient handling of game states and decision trees

| Layer      | Technology            |
| ---------- | --------------------- |
| Frontend   | HTML, CSS, JavaScript |
| Backend    | Python (Flask)        |
| AI Logic   | Minimax Algorithm     |
| Deployment | Render                |
